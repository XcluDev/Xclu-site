About

Xclu is a module-based creative IDE for media artists.
You create a program by combining modules and writing scripts.

Xclu is open source. It's made to help media artists and developers to do the following:
- (TODO) Generate and process video, sound and 3D graphics in real time or offline on CPU or GPU
- (TODO) Use ML and Computer Vision algorithms
- (TODO) Communicate with electronics using Arduino
- (TODO) Create compact stand-alone application which works stable 24/7

Supported platforms: Windows, Mac OS, Linux.
(TODO) Edge Computing Ready: NVidia Jetson Nano support.
